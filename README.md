# 🤖 Knight Bot

Ceci est un bot WhatsApp construit en utilisant la bibliothèque Baileys pour la gestion de groupe, incluant des fonctionnalités comme le tag de tous les membres, le mode muet/non-muet, et bien d'autres. Il est conçu pour aider les administrateurs à gérer efficacement les groupes WhatsApp.

## ⚙️ Fonctionnalités

- **Taguer tous les membres du groupe** avec la commande `.tagall`
- **Utilisation restreinte aux administrateurs** (Seuls les admins peuvent utiliser certaines commandes)
- **Jeux** comme le Morpion (Tic-Tac-Toe) pour l'engagement du groupe
- **Synthèse vocale** avec `.tts`
- **Création de stickers** avec `.sticker`
- **Détection anti-lien** pour la sécurité du groupe
- **Avertir et gérer les membres du groupe** avec le contrôle admin

---

## 📖 À Propos

Le Knight WhatsApp Bot aide les administrateurs de groupe en leur fournissant des outils pour gérer efficacement les grands groupes WhatsApp. Le bot utilise la bibliothèque Baileys pour interagir avec l'API WhatsApp Web et prend en charge les fonctionnalités multi-appareils.

Il est léger et peut être facilement personnalisé pour ajouter plus de commandes selon vos besoins. Le bot fonctionne dans un environnement Node.js et fournit une authentification basée sur un code QR ou un code de jumelage pour lier votre compte WhatsApp.

## 🛠️ Configuration & Installation

### Prérequis

- Node.js installé sur votre système
- Git installé (pour cloner le dépôt)

### Installation étape par étape

1. **Cloner le dépôt :**

    ```bash
    git clone https://github.com/votre-nom-utilisateur/knight-bot.git
    cd knight-bot
    ```

2. **Installer les dépendances :**

    ```bash
    npm install
    ```

3. **Lancer le bot :**

    ```bash
    node index.js
    ```

4. **Scanner le code QR :**

    Une fois le bot démarré, un code QR apparaîtra dans le terminal. Scannez ce code QR en utilisant la fonction "Appareils liés" dans WhatsApp pour connecter votre compte WhatsApp au bot.
